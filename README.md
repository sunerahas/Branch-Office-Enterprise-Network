# Branch-Office-Enterprise-Network
This project demonstrates a secure small-enterprise network design using Cisco Packet Tracer. The setup includes VLAN segmentation, Inter-VLAN Routing, wireless integration, and DHCP-based IP addressing for three departments within an organization: HR, IT, and Finance.
